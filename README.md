# TuneWave Backend

Бэкенд-платформа для **совместного прослушивания музыки в реальном времени**, комнатного чата, управления очередью треков и социальных механик (друзья, уведомления, баны).

## О проекте

**TuneWave** помогает пользователям:
- находить единомышленников по музыкальным интересам;
- слушать треки вместе в комнатах;
- ощаться в чате во время прослушивания;
- управлять очередью треков;
- получать рекомендации и вести историю активности.

Проект реализован на **FastAPI** и построен вокруг чистой архитектуры.
---

## Ключевые возможности

- OAuth-авторизация (Google / Spotify)
- Управление пользователями и профилями
- Комнаты (public/private), приглашения, роли участников
- Очередь треков и синхронизация воспроизведения
- Чат (HTTP + WebSocket)
- Избранные треки
- Дружба и заявки в друзья
- Уведомления
- Механизм банов
- Интеграции с Redis и RabbitMQ

## Технологический стек

### Backend
- Python 3.13

- Docker / Docker Compose

- Nginx (конфигурация присутствует в репозитории)

- Pytest / pytest-asyncio
- Ruff
- Mypy

## Структура проекта

```text
app/
  domain/          # Сущности, интерфейсы, доменные исключения
  application/     # Сервисы бизнес-логики и мапперы
  infrastructure/  # DB, брокеры, Redis, WebSocket-менеджеры
  presentation/    # API-роуты, схемы, middleware, auth
  config/          # Настройки, DI-контейнер, логирование
  main.py          # Точка входа FastAPI
tests/             # Тесты репозиториев и модулей
```
